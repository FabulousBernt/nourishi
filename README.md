# The Plateful

AI-powered recipe discovery and meal planning app. Find recipes by describing what you're craving, input what's in your pantry, or generate a full weekly meal plan — all with complete nutrition breakdowns.

Available at **[theplateful.app](https://theplateful.app)**

## How It Works

- **Recipe Search** — Describe what you want in natural language. The app searches both AI-generated and web-sourced recipes (via TheMealDB) to find matches.
- **Pantry Mode** — Enter ingredients you have on hand, get recipes that use what you've got.
- **Meal Planner** — Generate 1-week meal plans tailored to goals like bulking, cutting, low-carb, vegetarian, Mediterranean, or balanced eating.
- **Nutrition & Macros** — Every recipe and meal plan displays calories, protein, carbs, and fat with visual macro bars.
- **Export** — Download meal plans as PDFs or import them into your calendar.

Built with React 19, Next.js, and Cerebras AI. Runs as a PWA on any device. Deployed on Vercel with a serverless backend.

## Tech Stack

- **React 19 / Next.js** — UI framework
- **Cerebras AI** — AI-powered recipe generation and nutrition estimation
- **TheMealDB** — Web recipe database
- **Neon (Postgres)** — Community-curated recipe database with star ratings
- **Vercel** — Hosting & serverless functions
- **Capacitor 8** — Native iOS/Android compilation
- **Vitest** — Testing
